# OBS虚拟相机详细设置指南

## 为什么选择OBS虚拟相机？

OBS虚拟相机是最安全的采集方式，因为：
- ✅ 完全不注入游戏进程
- ✅ 不挂钩DirectX/OpenGL
- ✅ 只读取摄像头视频流
- ✅ 反作弊系统无法检测

---

## 详细设置步骤

### 1. 准备工作

确保你已经：
- 下载并安装了 OBS Studio
- 打开了腾讯麻将游戏并进入对局界面

---

### 2. 创建场景

1. 打开 OBS Studio
2. 点击左下角「场景」面板的 `+`
3. 输入场景名称，如「麻将」
4. 点击确定

---

### 3. 添加游戏源

#### 选项A：游戏捕获（推荐）

1. 在「来源」面板点击 `+`
2. 选择「游戏捕获」
3. 命名为「麻将游戏」
4. 在模式中选择「捕获特定窗口」
5. 窗口选择你的麻将游戏进程
6. 勾选「捕获光标」（可选）
7. 点击确定

#### 选项B：窗口捕获（如果游戏捕获失败）

1. 在「来源」面板点击 `+`
2. 选择「窗口捕获」
3. 选择你的麻将窗口
4. 窗口匹配优先级选择「窗口标题」
5. 点击确定

---

### 4. 调整画面

你应该能在OBS主预览区看到麻将画面了。

**重要：**
- 确保麻将画面占满OBS预览区
- 不要有黑边
- 保持16:9或4:3的比例

如果需要调整：
1. 在预览区右键点击游戏源
2. 选择「变换」→「拉伸到屏幕」

---

### 5. 启动虚拟相机

点击OBS主界面右侧的 `启动虚拟相机` 按钮。

如果按钮是灰色的，需要先安装虚拟相机插件：
- OBS Studio 28+ 自带虚拟相机
- 旧版本需要安装 OBS VirtualCam 插件

---

### 6. 验证虚拟相机

你可以使用Windows自带的「相机」应用测试：
1. 打开「相机」应用
2. 点击切换相机按钮
3. 应该能看到OBS的画面

---

### 7. 运行算牌器

```bash
python assistant.py --virtual-camera
```

---

## 常见问题

**Q: OBS里有画面，但虚拟相机是黑的？**

A: 确保没有选中「停用」来源，且虚拟相机已启动。

**Q: 虚拟相机在哪里选择？**

A: `assistant.py` 会自动检测，如有多个相机可能需要调整索引。

**Q: 画面延迟高？**

A: 在OBS「设置」→「输出」中降低FPS，或使用硬件编码。

---

## 高级设置

### 性能优化

在OBS「设置」→「输出」：
- 输出分辨率：1280x720
- 帧率：30 FPS
- 编码器：x264 或 NVIDIA NVENC

### 画面质量

在「设置」→「视频」：
- 基础分辨率：和游戏一致
- 输出分辨率：1280x720
- 缩小过滤器：Lanczos
